Baidu.com

If under $WEB_ROOT$/about/, then:

  1. Relative address based on current path, demo/ will redirected to $WEB_ROOT$/about/demo/.
  2. Absolute address based on this repo, /demo/ will redirected to $WEB_ROOT$/demo/.
  3. URL, https://www.baidu.com will redirected to https://www.baidu.com

If the url in not under this repo, be sure to use full address. For example, url www.baidu.com will redirected to $CUR_DIR$/www.baidu.com. Use https://www.baidu.com to set the correct link.
